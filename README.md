# student-details
This project is a menu-driven Student Management System developed in Python that focuses on storing, managing, and analyzing student academic records through a simple command-line interface. It allows users to enter student details such as name, roll number, and marks for three subjects and store them dynamically in a list structure, making it easy to perform multiple academic record operations efficiently. The program provides functionality to display all student data, search for a specific student using the roll number, update marks whenever required, calculate total marks and average automatically, assign grades based on predefined performance criteria, identify the class topper based on highest total marks, count the total number of students in the system, sort students according to their academic performance, calculate the overall class average, generate pass or fail lists based on average marks, check individual student grades quickly, and clear all stored records if needed. The project demonstrates practical implementation of fundamental Python programming concepts including loops, conditional statements, list manipulation, lambda expressions, sorting techniques, user input handling, and built-in functions such as sum(), max(), and len(), helping beginners understand how real-world data management systems work at a basic level. It also helps improve logical thinking, problem-solving ability, and familiarity with CLI-based applications while serving as a useful academic mini-project, portfolio project, or practice application for learning Python programming and software development fundamentals, and it can be further enhanced in the future by adding features such as file storage, graphical user interface support, database connectivity, authentication systems, advanced analytics, or web integration to transform it into a more complete student information management solution.
